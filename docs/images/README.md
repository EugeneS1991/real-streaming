# Images Directory

This directory contains images used in the project documentation.

## Usage in README.md

To insert an image in README.md, use one of the following formats:

### Relative Path (Recommended)

```markdown
![Alt text](docs/images/architecture-diagram.png)
```

### With Title

```markdown
![Architecture Diagram](docs/images/architecture-diagram.png)
```

### With Link

```markdown
[![Architecture Diagram](docs/images/architecture-diagram.png)](docs/images/architecture-diagram.png)
```

### Inline HTML (for more control)

```html
<img src="docs/images/architecture-diagram.png" alt="Architecture Diagram" width="800">
```

## File Naming Convention

- Use lowercase letters and hyphens: `architecture-diagram.png`
- Be descriptive: `load-balancer-setup.png`, `bigquery-schema.png`
- Use appropriate format: `.png` for screenshots, `.svg` for diagrams
